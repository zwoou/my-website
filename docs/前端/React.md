# React

## JSX基本语法

1. XML基本语法

   - 定义标签时，只允许被一个标签包括。
   - 标签一定要闭合。

2. 元素类型

   - DOM元素和组件元素，组件元素对应大写首字母
   - 注释
       使用{}包起来
   - DOCTYPE

3. 元素属性
   DOM元素的属性是标准规范属性，但有两个意外--class和for

   - class属性改为className
   - for属性改为htmlFor
   - Boolean属性
      省略Boolean属性会默认设置为true
   - 展开属性
4. JavaScript属性表达式
   属性值要使用表达式，只要用 {} 替换 "" 即可

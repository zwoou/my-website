# ESLint 入门

[ESLint 入门 - ESLint - 插件化的 JavaScript 代码检查工具](https://zh-hans.eslint.org/docs/latest/use/getting-started)

## 安装并使用

```shell
npm init @eslint/config
```

## 配置

在目录下有`.eslintrc.{js,yml,json}` 文件

- vscode插件配置
  
  ```
      "editor.codeActionsOnSave": {
          
          "source.fixAll": "explicit"
      }
  ```
